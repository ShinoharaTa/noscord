# Noscord

**Nostr プロトコルを使用したパブリックなチャンネル型チャットクライアント**です。  
SvelteKit + Vite で動作します。

## ドキュメント

詳細は `docs/README.md` にまとめています。

## 開発

```bash
npm install
npm run dev
```

型チェック/ビルド/リント:

```bash
npm run check
npm run build
npm run lint
```
